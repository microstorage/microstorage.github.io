KINETIC SYSTEM Il nuovo concetto modulare Kinetic system è costituito da una stazione base che può essere ampliata utilizzando diversi moduli aggiuntivi. I singoli moduli possono essere combinati individualmente a seconda degli obiettivi di allenamento o semplicemente per le dimensioni del vostro locale. MODULO 3 – LEG PRESS Modulo addizionale per stazione base. Stazione base estendibile utilizzando singolarmente o il modulo 3 o il modulo 2 Perfetto allenamento per i muscoli delle gambe e glutei mediante la struttura a parallelogramma ergonomica Dimensione in uso Stazione base + modulo 3: 176 / 246 / 215 cm 160 kg di carico massimo Peso max utilizzatore: 150 Kg ATTENZIONE: IL PREZZO SI RIFERISCE SOLO AL MODULO. LA STAZIONE BASE KINETIC SYSTEM NON È COMPRESA NEL PREZZO.
